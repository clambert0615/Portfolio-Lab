POS Project

This project was a group project with myself and two other students. The goal of the project was to develop a point of sale system for a retailer. The system asks the user which products and the quantity they would like to buy and adds them to the cart. The system then asks the user how they would like to pay. If the user selects credit card, the system has many validation techniques to make sure that the credit card is valid. The system also has an administrative section, where the store employees can enter in new products and delete products that are no longer sold. The inventory list is stored into a text file that is updated and read each time the program starts. This project took us about two days and also included a lot of time spent learning how to use github branching.
